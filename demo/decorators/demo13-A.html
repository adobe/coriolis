Store Decorator

Description

This demo demonstrate the api of the store decorator.

Code

    var coriolis = Coriolis.createIframe(
      domElement,
      iframeUrl,
      initialContent
    );
    coriolis.store.set(name, value);
    coriolis.store.set(keyValueObject);
    coriolis.store.get(name);
    coriolis.store.on(name, callback);
    coriolis.store.once(name, callback);
